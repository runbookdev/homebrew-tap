# Homebrew Tap for runbook

Executable runbooks as code — documentation and automation in a single file.

## Install
```bash
brew install runbookdev/tap/runbook
```

## Upgrade
```bash
brew upgrade runbook
```

## More info

- Main repo: [github.com/runbookdev/runbook](https://github.com/runbookdev/runbook)
- Documentation: [runbookdev.github.io/docs](https://runbookdev.github.io/docs)
